📡 Universal Stream Channel Parser

How to use:
This page expects data via URL query parameters. No manual input.
?url=<url>&parser=<type> – fetches content from the URL and parses.
?data=<content>&parser=<type> – parses the raw content directly.
?parser=<type> – uses the fallback sample.
Supported parsers: m3u, json, webpage, direct, iframe, xml, rss, xspf, csv, pls, text.

No input fields – all data from URL | CORS may block external fetches; use data parameter instead.
